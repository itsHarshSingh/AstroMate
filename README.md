# AstroMate App

AstroMate is a simple Android application built using Java in Android Studio.  
It provides basic astrology features like horoscope and name-based predictions.

## Features
- Simple and clean user interface
- Name-based astrology information
- Easy to use design
- Beginner-friendly project
- Share astrology results with friends via text message

## Tech Stack
- Java
- Android Studio
- XML (UI Design)

## Purpose
This project is made for learning Android development and practicing basic app building concepts.
